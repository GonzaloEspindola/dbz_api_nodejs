# Dragon Ball API

API en node.js de la popular serie "Dragon Ball".

## Getting Started
Check out the [documentation][1] to get started

## Dependencies

```bash
npm install cors@2.8.5
npm install doteenv@16.0.1
npm install express@4.18.1
npm install pg@8.7.3
```
