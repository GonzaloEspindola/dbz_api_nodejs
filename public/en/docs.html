REST

Base url:

https://dragon-ball-super-api.herokuapp.com/api

The url contains the information about the available resources of the API. All requests will return json data.

GET https://dragon-ball-super-api.herokuapp.com/api
                        
{
    "characters": "https://dragon-ball-super-api.herokuapp.com/api/characters"
}

Available resources:

  • Characters: get all characters.

Information

This documentation will help you become familiar with he Dragon Ball API resources by showing you the different daa queries.

Character schema

Key Type Description
id int The id of the character
imageUrl string (url) An image (.png) of the character
name string The name of the character
originplanet string The origin planet of the character
role string The most important rol or function of the character
specie string The specie of the character
status string The status of the character
transform array Array with character transformations (if any)
universe string Universe to wich the character belongs

Get all characters

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters

You can access the list of all characters using the endpoint /characters.

GET https://dragon-ball-super-api.herokuapp.com/api/characters
                        
[
    {
        "id": 124,
        "name": "Daishinkan",
        "specie": "Angel",
        "role": "Sumo Sacerdote",
        "imageUrl": "https://i.ibb.co/cY94Ddp/Daishinkan.png",
        "status": "Vivo",
        "universe": "0",
        "transform": [],
        "originplanet": "Desconocido"
    },
    ...
]

Simple pagination

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters?limit={max-characters}

You can access a certain pagination using the parameter limit /characters?limit={max-characters}. This will return an array with the characters separated based on the parameter limit

GET https://dragon-ball-super-api.herokuapp.com/api/characters/characters?limit=50
                        
[
    0: {...}
    1: {...}
    2: {...}
]
                        
// Array of 50 characters each position.

Offset y limit

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters?offset={x}?limit={y}

You can access a certain pagination using the query parameters offset y limit /characters?offset={x}?limit={y}.

GET https://dragon-ball-super-api.herokuapp.com/api/characters/characters?offset=0?limit=50
                        
[
    0: {...}
    1: {...}
    2: {...}
]

Get a single character by name

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters/{name}

You can get a single character using the name /characters/{name}.

GET https://dragon-ball-super-api.herokuapp.com/api/characters/Gohan
                        
{
    "id": x,
    "imageUrl": "https://i.ibb.co/XjmRwZp/Dragon-Ball-Gohan-PNG-Pic.png",

    "role": "Guerrero Z",
    "specie": "Saiyan Mestizo",
