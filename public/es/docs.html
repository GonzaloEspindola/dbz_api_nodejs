REST

URL base:

https://dragon-ball-super-api.herokuapp.com/api

La url base contiene la informacion sobre los recursos disponibles de la API. Todas las solicitudes devolveran datos json.

GET https://dragon-ball-super-api.herokuapp.com/api
                        
{
    "characters": "https://dragon-ball-super-api.herokuapp.com/api/characters"
}

Recursos disponibles:

  • Personajes: obtener todos los personajes.

Informacion

Esta documentacion te ayudara a familirizarte con los recursos de la API de Dragon ball mostrandote las diferentes consultas de datos.

Esquema de personaje

Key Tipo Descripcion
id int El id del personaje
imageUrl string (url) Una imagen (.png) del personaje
name string El nombre del personaje
originplanet string El planeta de origen del personaje
role string El rol mas importante o actual que cumple el personaje
specie string La especie del personaje
status string Estado actual del personaje
transform array Array con las transformaciones del personaje (si tiene)
universe string Universo al cual pertenece ese personaje

Obtener todos los personajes

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters

Puedes acceder a la lista de todos los personajes usando el endpoint /characters.

GET https://dragon-ball-super-api.herokuapp.com/api/characters
                        
{
    "characters": "https://dragon-ball-super-api.herokuapp.com/api/characters"
}

Paginacion simple

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters?limit={max-characters}

Puedes acceder a cierta paginacion utilizando el parametro query limit /characters?limit={max-characters}. Esto devolvera un array con los personajes separados en objetos separados en base al parametro limit

GET https://dragon-ball-super-api.herokuapp.com/api/characters/characters?limit=50
                        
[
    0: {...}
    1: {...}
    2: {...}
]
                        
// Array de 50 personajes cada posicion.

Offset y limit

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters?offset={x}?limit={y}

Puedes acceder a cierta paginacion utilizando el parametro query offset y limit /characters?offset={x}?limit={y}.

GET https://dragon-ball-super-api.herokuapp.com/api/characters/characters?offset=0?limit=50
                        
[
    0: {...}
    1: {...}
    2: {...}
]

Obtener un personaje por nombre

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters/{name}

Puedes acceder a un personaje usando su nombre /characters/{name}.

GET https://dragon-ball-super-api.herokuapp.com/api/characters/Gohan
                        
{
    "id": x,
    "imageUrl": "url",
    "name": "Gohan",
    "originplanet": "Planeta Tierra",
    "role": "Guerrero Z",
    "specie": "Saiyan Mestizo",
    "status": "Vivo",
    "transform": [],
    "universe": "7"
}
